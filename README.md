# 🌿 **Multimodal LLM-Based Recommendation System**

### *A Fine-Tuned LLM for Leaf Disease Detection and Treatment Recommendations*

This project leverages a **fine-tuned multimodal Large Language Model (LLM)** to detect, classify, and provide treatment recommendations for leaf diseases with high accuracy.

---

## 🚀 **Key Features**

- **Leaf Disease Detection & Classification**:  
  - Fine-tuned **LLaVA (Large Language-and-Vision Assistant)** on a custom dataset.  
  - Achieved **90% accuracy** in detecting and classifying leaf diseases.  

- **Metric-Assisted Experiments**:  
  - Evaluated LLM applications and conducted experiments to ensure **dependability** and **reproducibility**.  

- **AI Chat-Bot**:  
  - Developed a chatbot application to:  
    - Provide **treatment recommendations** for detected leaf diseases.  
    - Facilitate **follow-up interactions** for ongoing care.  

---

## ⚙️ **Tech Stack**
- **Programming Language**: Python  
- **LLM Framework**: LLaVA (Fine-tuned)  
- **Tools**: Langflow for workflow orchestration  

---

## 🎯 **Outcomes**
- 90% accuracy in disease classification.  
- Reliable and reproducible LLM experiments.  
- Intuitive chatbot for actionable recommendations and interactive user support.  

---

**This system is designed to empower farmers and agronomists with AI-driven solutions for plant health monitoring and treatment.** 🌱
